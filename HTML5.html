Q.2 How to embed audio and video in a webpage ?

Create a new HTML file in the same directory, called index.html . Add tag audio and tag video elements to the page; make them display the default browser controls. Give both of them tag source elements so that browsers will find the audio format they support best and load it. These should include type attributes.
